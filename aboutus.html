About YU Marje3

Hi there! Welcome to YU Marje3, a conceptual academic resource website that I, Sarah Salameh, designed and built as a final project for my CIS 201: Fundamentals of Web Design course.

Our Goal

YU Marje3 aims to be a friendly, organized, and accessible online platform. It makes finding study materials—like lecture notes, slides, and text books—much easier for students. Every element reflects the skills I acquired during the course.

Design & Technology

This site was built with clean code and user experience in mind, using:

  • HTML (Structure): Structures all content logically.
  • CSS (Style & Layout): Handles style and layout, making it fully responsive for any device.
  • JavaScript (Interactivity): Adds interactivity, including Dark/Light mode and dynamic navigation features.

Get Involved

Your feedback is highly valued! You can contribute by:

  • Submitting Content: Notes, guides, lectures for the archive.
  • Reporting Issues: Bugs, broken links, or visual glitches.
  • Giving Feedback: Constructive critique on design or usability.
